##Media-Script
Здравствуйте, мои маленькие любители контента! Этот репозиторий предоставляет возможность загрузки аудио и видео материалов из сети Интернет. Кроме того, он предлагает функции обрезки и конвертации аудио/видео файлов, а также загрузки превью и текстовых описаний видео с YouTube, также есть функционал для транскрибации видео.
#Установка зависимостей
Для начала работы с репозиторием необходимо установить две библиотеки: ffmpeg [https://github.com/FFmpeg/FFmpeg] и yt_dlp [https://github.com/yt-dlp/yt-dlp]. После установки этих библиотек, их следует добавить в переменные окружения.
После выполнения этих шагов, откройте проект и перейдите в файл main.py. С помощью терминала установите все необходимые зависимости, выполнив команду pip install -r requirements.txt.
